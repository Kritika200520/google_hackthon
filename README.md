# Assistive Vision System (Google Hackathon)

An AI-powered wearable assistant for the visually impaired using Arduino, ESP32, and Google Gemini AI.

## 🚀 Features
- **AI Scene Analysis:** Uses ESP32-CAM and Gemini to describe the environment.
- **Haptic Navigation:** L298N-driven vibration feedback for obstacle detection.
- **Low-Cost Hardware:** Utilizes jumper-wire sensors and repurposed components.
- **Emergency Alerts:** Automated SOS based on fall detection.

## 🛠️ Tech Stack
- **Hardware:** Arduino, ESP32-CAM, L298N, Ultrasonic Sensors.
- **Software:** C++, Arduino IDE, Google Gemini API.

## 🔧 Setup
1. Clone the repo.
2. Input your WiFi SSID and Gemini API Key in `config.h`.
3. Flash the Arduino and ESP32 modules.
